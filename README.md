# Geogeo

Geogeo uses data from WhosOnFirst and try its best to normalize the results. I use it for my own needs so some data may be missing for you, feel free to open an issue if you want to add something to the API

## Dependencies

Geogeo uses Postgis, that's it.

## Import

Run `bin/rake localities:import_wof`

## API

```
curl "http://localhost:3000/localities?latitude=48.8534&longitude=2.3488"
```
